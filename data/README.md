# Data Directory

This local directory is for lightweight manifests and derived tables only.

Raw xBD satellite images remain on the remote host and should not be copied into the local submission-material root unless license terms are explicitly checked.

Expected outputs:

- `derived/xbd_buildings.csv`
- `derived/xbd_label_files.csv`
- `derived/xbd_event_summary.csv`
- `manifests/auxiliary_data_sources.csv`

For data access, licensing, and redistribution boundaries, see:

- `reports/data_access_license_notes.md`
